# AI-Assistant-Chatbot-n8n
AI chatbot workflow built using n8n, Gemini, OpenRouter and SerpAPI.

## Overview
This project is an AI-powered chatbot workflow built using n8n with LLM integration and tool support.

## Features
- AI Agent
- Conversation Memory
- Google Gemini Integration
- OpenRouter Integration
- Calculator Tool
- Date & Time Tool
- SerpAPI Web Search

## Technologies Used
- n8n
- Google Gemini
- OpenRouter
- SerpAPI

## Workflow Description
The chatbot receives user messages through a chat trigger and processes them using an AI Agent connected with memory and multiple tools.

## Tools Included
- Calculator
- Date & Time
- Web Search

## Importing Workflow
1. Download chatbot.json
2. Open n8n
3. Import the workflow
4. Add your own API credentials
5. Run the workflow

## Project Type
AI Automation / AI Agent Workflow
